# 概要

さいころをグリッド上で転がす謎解きの訓練用アプリケーション

# リンク

[https://f-mm7.github.io/dice-trainer/](https://f-mm7.github.io/dice-trainer/)

# ルール

立方体がグリッド上を経路(colered line)に沿って転がるとき,
初期位置(赤端)と同じ面で接地するマスを全て選択し ENTER を押す.

CLEAR を押すと選択が解除される.
